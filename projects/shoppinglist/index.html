Shopping List Application


Overview


Objective: Develop a real-time shopping list app to simplify group shopping tasks by reducing the need for calls or texts. Users can create, update, and share lists collaboratively on a shared platform.


Role: Lead Developer and Scrum Master for the React Native app. Designed and created prototypes in Figma. Responsible for over 90% of the app's code, architecture, and design.


Duration: September 2024 — December 2024


Team Size: React Native App Development: 3 members. Figma Prototyping: 3 members.


Tech Stack


  • Frontend: React Native, Expo
  • Backend: Firebase Firestore, Firebase Cloud Messaging
  • Storage: AsyncStorage (for storing user login information)
  • Environment Management: react-native-dotenv
  • CI/CD: Expo EAS
  • Design Tools: Figma

Key Features


  • User Management: Simple username-based login (aligned with product owner's security preference).
  • Real-Time Collaboration: Users can share and edit shopping lists via invitation codes and usernames.
  • Auto-Sorting: Automatically organizes list items by completion status and alphabetical order.
  • Offline Functionality: Local data storage using AsyncStorage, enabling auto-login.
  • Responsive Design: Reusable UI components ensure consistency and maintainability.

Achievements


  • Designed reusable components, reducing code redundancy by 20%.
  • Achieved a 95% crash-free rate through rigorous testing and debugging.
  • Improved usability by 30% through clean, intuitive UI and enhanced navigation.
  • Successfully delivered both the React Native app and Figma prototype within the project timeline.
