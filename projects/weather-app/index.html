Weather Application


Overview


Objective: Develop a mobile application to provide users with real-time weather information and forecasts.


Duration: November 2023 — December 2023


Team Size: Solo project


Tech Stack


  • Frontend: React Native
  • Backend: OpenWeatherMap API
  • Design Tools: Expo, React Native Paper

Key Features


  • Current Weather: Display current weather information including temperature, wind speed, and weather description.
  • Weather Icons: Display weather icons based on the current weather condition.
  • Location Input: Allow users to manually input a location to get the weather information.
  • Daily Forecast: Provide a daily weather forecast.
  • About Page: Provide information about the app.
  • Help Page: Provide useful links and information.

Achievements


  • Real-Time Data: Provided users with up-to-date weather information.
  • User-Friendly Interface: Developed an intuitive interface for easy navigation and weather information retrieval.
  • Cross-Platform Support: Ensured the app works seamlessly on both Android and iOS devices.
