Time Tracking Application


Overview


Objective: Develop a web application to help users track time spent on different activities, manage tasks, and analyze time usage.


Duration: September 2023 — November 2023


Team Size: Solo project


Tech Stack


  • Frontend: ReactJS, Vite
  • Backend: JSON Server
  • Charts: Chart.js
  • Design Tools: CSS, React Beautiful DnD

Key Features


  • Task Management: Create, edit, and delete tasks with customizable tags and order.
  • Time Tracking: Start and stop time tracking for tasks, with support for single-task mode.
  • Time Analysis: Analyze time spent on tasks within a specified interval and view detailed reports.
  • Activity Charts: Visualize daily activity times for selected tasks using bar charts.
  • Responsive UI: Supports various screen resolutions with light and dark theme options.
  • Drag and Drop: Reorder tasks using drag-and-drop functionality.

Achievements


  • Enhanced Productivity: Provided users with tools to efficiently manage and track their time, improving productivity.
  • Intuitive Interface: Developed a user-friendly interface that simplifies task management and time tracking.
  • Data Visualization: Enabled users to visualize their time usage through interactive charts, aiding in better time management.
  • Scalable Codebase: Ensured the codebase is maintainable and scalable by following best practices in ReactJS development.
