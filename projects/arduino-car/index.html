Autonomous Arduino Car


Overview


Objective: Develop an autonomous robot control system using Arduino, ESP8266, and MQTT for communication, featuring LIDAR for distance sensing, a compass for orientation, and PWM for motor control.


Duration: January 2022 — April 2023


Team Size: Group of 3 (I was responsible for over 90% of the project)


Tech Stack


  • Microcontroller: Arduino
  • Communication Module: ESP8266
  • Protocol: MQTT
  • Programming Languages: C++, Arduino
  • Data Format: JSON

Key Features


  • Distance Sensing: Utilizes LIDAR for accurate distance measurement.
  • Orientation: Integrates a compass module for determining the robot's direction.
  • Motor Control: Uses PWM signals for precise motor control.
  • Wireless Communication: Employs ESP8266 for WiFi connectivity and MQTT for remote command processing and telemetry.
  • Manual Operation: Features a joystick-based control system for manual operation.
  • Data Transmission: Formats data in JSON for efficient transmission.

Achievements


  • Enhanced Automation: Developed a fully autonomous robot capable of navigating and avoiding obstacles.
  • Improved Communication: Implemented reliable wireless communication using MQTT, enabling remote control and monitoring.
  • Accurate Sensing: Achieved precise distance and orientation measurements using LIDAR and compass modules.
  • Efficient Control: Ensured smooth and precise motor control using PWM signals.
