Hotel Room Booking Program


Overview


Objective: Develop a simple yet comprehensive hotel room booking system suitable for users with basic computer knowledge.


Duration: September 2021 — December 2021


Team Size: Solo project


Note: This project was my first programming project after learning my very first programming language, C++. It may not show advanced techniques, but it demonstrates my potential and growth as a programmer.


Tech Stack


  • Language: C++
  • Libraries: Standard C++ Libraries

Key Features


  • Show Prices of Room Types: Allows users to view the prices of different types of rooms available in the hotel.
  • Room Reservation: Users can book rooms of their choice.
  • Show Available Rooms: Displays the number of rooms available for each room type.
  • Search Customer: Enables searching for customers by reservation number or name.
  • Input Validation: Prevents invalid inputs from crashing the program.
  • Random Number Generation: Used for generating room numbers and reservation numbers.
  • Discount Offers: Users have a chance to win a discount on room prices.

Achievements


  • Comprehensive Functionality: Implemented a full-featured booking system with various functionalities.
  • User-Friendly Interface: Developed an intuitive interface for easy navigation and use.
  • Robust Input Validation: Ensured the program handles invalid inputs gracefully.
  • Efficient Random Number Generation: Utilized random number generation for unique room and reservation numbers.
